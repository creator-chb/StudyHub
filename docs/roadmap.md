# StudyHub 项目发展规划

## 📋 总体目标

将 StudyHub 从一个纯前端的轻量级工具，逐步演进为功能完善的学习管理平台。

---

## 🎯 版本规划

### v1.4.0 - 架构重构与功能增强

**发布时间：2026 年 3 月 20 日**

1. **代码架构重构** 🏗️
   - [x] 单文件拆分为模块化结构
   - [x] 使用 IIFE 模式封装模块
   - [x] 消除全局变量污染
   - [x] 完整的 JSDoc 类型注释

2. **功能增强** ✨
   - [x] 深色模式支持（浅色/深色/跟随系统）
   - [x] 数据导入导出功能（JSON格式）
   - [x] 链接分类/标签系统
   - [x] 任务优先级设置（高/中/低）

3. **用户体验优化** 📱
   - [x] 移动端触摸操作优化（滑动删除）
   - [x] 键盘导航增强
   - [x] 任务搜索和筛选功能

4. **技术债务清理** 🔧
   - [x] 移除内联事件处理器
   - [x] 统一错误处理机制
   - [x] 配置中心化管理
   - [x] 插件化架构基础

---

### v1.3.0 - 交互优化与移动端适配

**发布时间：2026 年 3 月 20 日**

1. **移动端适配** 📱
   - [x] 小屏下模态框宽度自适应、内边距缩小
   - [x] 链接网格最小列宽自动调整
   - [x] 任务关联链接 Chips 换行优化

2. **快捷操作** ⌨️
   - [x] 全局快捷键：N 新建链接，T 新建任务，/ 聚焦搜索框
   - [x] 列表项键盘导航（上下键选择，Enter 打开，Delete 删除）
   - [x] 链接搜索功能

3. **链接置顶** 📌
   - [x] 支持将常用链接置顶显示
   - [x] 置顶链接视觉区分（金色边框）
   - [x] 一键置顶/取消置顶
