# StudyHub 项目发展规划

## 📋 总体目标

将 StudyHub 从一个纯前端的轻量级工具，逐步演进为一个功能完整的前后端分离的学习管理平台。

---

## 🎯 近期规划

### v1.3.0 - 交互优化与无障碍（计划中）

**预计发布时间：2026 年 4 月上旬**

1. **交互细节提升** 🎯
   - [ ] 一键跳转改为"新标签页分批打开"，显示进度提示（已打开 x/共 y）
   - [ ] 链接卡片增加"复制 URL"按钮
   - [ ] 拖拽排序功能（链接池和任务列表）
   - [ ] 支持"置顶"常用项

2. **可访问性（a11y）** ♿
   - [ ] 模态框焦点陷阱（Focus Trap），Tab 循环聚焦
   - [ ] 关闭后焦点返回触发按钮
   - [ ] 补充 aria-label/aria-describedby
   - [ ] 错误提示不只依赖颜色（图标 + 文字）

3. **移动端适配** 📱
   - [ ] 小屏下模态框宽度自适应、内边距缩小
   - [ ] 链接网格最小列宽自动调整
   - [ ] 任务关联链接 Chips 换行优化

---

### v1.4.0 - 工程化与功能扩展（计划中）

**预计发布时间：2026 年 4 月下旬 - 5 月**

1. **工程化改进** 🔧
   - [ ] 代码模块化重构（renderLinksView、renderTasksView 等独立模块）
   - [ ] 使用 JSDoc 为数据结构与关键函数加类型注释
   - [ ] 统一工具函数（日期格式化、URL 校验、转义）放入 utils 模块
   - [ ] 内联 JS 字符串拼接改为事件监听与 dataset 传参

2. **安全加固** 🛡️
   - [ ] URL 解析 hostname，屏蔽可疑协议（data://、javascript://）
   - [ ] 渲染层转义优化

3. **搜索与过滤** 🔍
   - [ ] 链接池按名称/域名搜索
   - [ ] 任务按名称/状态（进行中、已完成、逾期）过滤
   - [ ] 简单标签筛选（预备标签系统）

4. **主题与个性化** 🎨
   - [ ] 明暗色主题切换（CSS 变量统一管理）
   - [ ] 记住用户偏好到 localStorage
   - [ ] 自定义强调色与卡片密度（紧凑/舒适）

5. **快捷操作** ⌨️
   - [ ] 全局快捷键：N 新建链接，T 新建任务，/ 聚焦搜索框
   - [ ] 列表项键盘导航（上下键选择，Enter 打开，Delete 删除）

## 🎯 长期规划：v2.0 全栈版本

**预计发布时间：2026 年 Q4**

### 技术栈选型

**前端：**
- Vue.js 3 / React + Vite
- UI 组件库：Element Plus / Ant Design
- 状态管理：Pinia / Redux
- 路由：Vue Router / React Router

**后端：**
- Node.js + Express/Fastify
- Python + FastAPI/Django
- Go + Gin（备选）

**数据库：**
- MongoDB（文档型，灵活）
- MySQL/PostgreSQL（关系型，规范）
- SQLite（轻量级，初期使用）

**认证方案：**
- JWT Token
- Session/Cookie
- OAuth 2.0（第三方登录）

**部署：**
- Docker + Docker Compose
- CI/CD 流程
